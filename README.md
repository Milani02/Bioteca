# 🎥 Bioteca | Videoteca Corporativa

Uma plataforma moderna e interativa desenvolvida para gerenciar e hospedar vídeos e treinamentos corporativos. O projeto foca em entregar uma experiência de usuário fluida, com uma interface limpa e animações imersivas.

## ✨ Destaques do Projeto

- **Catálogo de Treinamentos:** Organização intuitiva de vídeos e trilhas de capacitação corporativa.
- **Interface "Liquid Glass":** Design moderno utilizando conceitos de glassmorphism para um visual limpo e profissional.
- **Animações Fluidas:** Transições de tela e interações de elementos suaves para melhorar a experiência do usuário.
- **Gestão de Acesso:** Sistema estruturado para garantir que os colaboradores acessem os conteúdos corretos de forma segura.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção da interface de usuário.
- **[Vite](https://vitejs.dev/)** - Ferramenta de build super rápida para o frontend.
- **[Supabase](https://supabase.com/)** - Backend as a Service (BaaS) utilizado para banco de dados, autenticação e armazenamento.
- **[Framer Motion](https://www.framer.com/motion/)** - Biblioteca principal para a criação das animações avançadas e efeitos visuais da interface.
- **[TypeScript](https://www.typescriptlang.org/)** - Adicionando tipagem estática para um código mais seguro e escalável.
